# PW-Core Workspace

This project has been converted to TypeScript completely and configured to run with a local fake DB + API setup.

## Features
- Fully client-side persisted Database (Projects, Tasks, Activities, Documents) in `localStorage`.
- Fake Authentication module (supports email/password mock login, register with OTP validation, etc.).
- Complete local styling and responsive dashboard.

## Development

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open the browser to the local URL (usually `http://localhost:5173`).
